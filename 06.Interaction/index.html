Interactions

JavaScript has different functions to interact with the user through the browser, which are discussed in this chapter.

alert

The alert function sends a pop up with a message, that waits for the user to click "OK"

More article text (if needed)

prompt

The prompt is a modal window, just like the alert, but it accepts two arguments.

The modal window asks the user for some input, which can be typed.

confirm

The confirm function allows us to ask the user a question through a modal window, accepting OK or Cancel for an answer.

Pressing OK will make the result true and pressing Cancel will make it false.
